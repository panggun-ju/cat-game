# Cat Dropping Things

냥이가 물건을 밀어버리는 게임입니다.

## 게임 특징

- **픽셀 아트 스타일**: 귀여운 고양이와 집안 풍경
- **물리 기반**: 고양이가 물건을 밀어버리는 재미있는 물리 시뮬레이션
- **다양한 오브젝트**: 책상 위 물건들을 밀어버리세요

## 필요한 것

- Python 3.x
- Pygame 라이브러리

## 설치 방법

1. **Python 설치**
   - [python.org](https://python.org)에서 Python 3.x 다운로드 및 설치

2. **Pygame 설치**
   ```
   pip install pygame
   ```

## 실행 방법

1. 이 폴더를 열고
2. 터미널(명령 프롬프트)에서 다음 명령 입력:
   ```
   python game.py
   ```

## 조작법

- **마우스**: 고양이가 마우스를 따라갑니다
- **클릭**: 고양이가 물건을 밀어버립니다

## 파일 설명

| 파일 | 설명 |
|------|------|
| `game.py` | 메인 게임 파일 |
| `cat.png` / `cat.json` | 고양이 스프라이트 |
| `objects.png` / `objects.json` | 물건 이미지들 |
| `background.png` | 배경 이미지 |
| `push.wav` / `fast_push.wav` | 효과음 |
